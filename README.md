## ✨ Возможности

- 🔍 **Поиск событий** через KudaGo API
- 🔐 **Аутентификация** (Email/Google OAuth)
- 📱 **5 вкладок** (Explore, Events, Favorites, Map, Profile)
- ❤️ **Избранное** с синхронизацией
- 🎨 **Темная/светлая тема**
- 🔔 **Уведомления** о событиях

## 🛠️ Технологии

- **SwiftUI** + **MVVM** архитектура
- **Firebase** (Auth, Firestore, Storage)
- **Google Sign-In** OAuth
- **URLSession** + **Async/Await**
- **PhotosUI**

## 🚀 Быстрый старт

1. Клонируйте репозиторий
2. Откройте `EventHub.xcodeproj`  
3. Установите зависимости (SPM)
4. Настройте Firebase (`GoogleService-Info.plist`)
5. Запустите проект ⌘+R
